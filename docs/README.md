# Dashboard de Análisis de Ahorro Energético

## Descripción
Aplicación Java para análisis y visualización de datos de ahorro energético provenientes del sistema SGC.

## Características
- ✅ Carga y procesamiento de datos CSV
- ✅ Dashboard interactivo con gráficos
- ✅ Modelo de Machine Learning para predicciones
- ✅ Análisis estadístico automático
- ✅ Exportación de resultados

## Instalación

### Requisitos
- Java 11 o superior
- Maven 3.6+
- Python 3.8+ (para scripts)

1. Clonar el repositorio
